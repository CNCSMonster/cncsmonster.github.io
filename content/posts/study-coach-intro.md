## 一句话

**你问它一个问题，它不问回来就不算完。**

study-coach 是一个 agent skill。触发后，AI 会切换成苏格拉底式教学模式——不直接给答案，而是从你的直觉出发、问问题、等你接、再补全。教到你真的会。

## 和普通 AI 问答的区别

**问题：** "英雄之旅的拒绝召唤阶段是做什么的？"

**普通 AI：**
> 拒绝召唤是英雄之旅的第二个阶段。主角接到冒险召唤后因为恐惧或犹豫而拒绝出发，这个阶段用来展现主角的弱点…

**study-coach：**
> 用《狮子王》辛巴的例子来看——他拒绝回荣耀石，不是怂，是在造势。你先感受一下：如果你看一部电影主角听到召唤二话不说就走了，观众的感觉和辛巴拒绝后再走，有什么不同？
> 
> 然后再画回箭头图——你把刚才的感受落到结构上："触发事件 → 召唤 → 拒绝 → 导师 → 出发"。拒绝卡在中间，让恐惧先被承认，出发才有重量。
> 
> 那《献给阿尔吉侬的花束》里的查理接受手术前几乎没有抗拒——为什么你不会觉得"缺少"拒绝召唤？

它不是不给你知识。它让你自己走最后一步推理链。

## 能学什么

| 领域 | 你会学到 |
|------|---------|
| 技术 | 为什么 Git rebase 会改变 commit hash？SHA1 的输入字段之一是父 commit 的 hash — rebase 换基线等于换父 |
| 文学 | 拒绝召唤在叙事结构里做什么用——它让观众先和角色一起评估风险，出发才有重量 |
| 物理 | 失重不是没重力，是支持力为零。宇航员在 400km 高空还有 90% 重力——他们飘是因为轨道就是自由落体 |
| 思维 | 第一性原理 vs 类比思维——Musk 不问"火箭一般多少钱"而问"铝多少钱一公斤" |

不限领域，不限体量。大到系统性学叙事结构，小到"为什么负数乘负数得正"都可以。

## 怎么触发

只要说这些话，AI 就会切到 study-coach 模式：

```
教我 X
讲讲 X
我想学 X
搞懂 X
帮我理解 X
```

## 什么时候它不会插嘴

```
帮我写一个用 Promise 调接口的函数    → 只给代码，不给教案
Promise.all 的语法是什么？           → 直接回答语法
帮我 debug 这段代码                  → 直接排查问题
```

**设计原则：你说教才教，不猜你要不要。**

## 为什么这样设计

大多数 AI 的教学方式是"我知道答案，我告诉你"。这对查用法有用，对学概念不太有用——接收不等于理解。

study-coach 用 6 条原则把对话变成"思维脚手架"：你的直觉被接住、偏差被校准、推理链最关键的一步你亲手走完。你不是在听讲，是在自己推。
