    这本电子书是笔者学习 Learn Claude Code 项目的中文学习笔记。

    Learn Claude Code 是一个学习项目，帮助开发者理解 Claude Code 的核心概念：

    The model is the agent. The code is the harness.

    笔记采用 QA 形式 整理每个 session 的核心概念，包含：

    • 核心概念 - Agent Loop、Tool Use、TodoWrite 等
